# Game Screenshot OCR

게임 스크린샷에서 아이템/장비 정보를 추출하는 웹 애플리케이션입니다.
브라우저에서 동작하며, 서버에 이미지가 전송되지 않습니다.

**https://jin54.github.io/ocr/**

## 주요 기능

- **이미지 업로드** - 드래그앤드롭 또는 클릭으로 여러 이미지 업로드
- **영역 선택** - 이미지 위에 OCR 대상 영역을 직접 그려서 지정
- **한국어/영어 OCR** - Tesseract.js 기반 클라이언트 사이드 텍스트 인식
- **편집 가능한 테이블** - 추출 결과를 테이블에서 바로 수정 가능
- **데이터 내보내기** - CSV, Excel, JSON 형식으로 다운로드
- **영역 템플릿** - 자주 쓰는 영역 배치를 저장/불러오기

## 사용법

1. 게임 스크린샷을 업로드합니다
2. "영역 그리기" 버튼을 누르고, 텍스트를 추출할 부분을 드래그합니다
3. 각 영역에 이름을 지정합니다 (예: 아이템명, 레벨, 스탯)
4. "OCR 실행" 버튼을 누르면 결과가 테이블에 표시됩니다
5. 여러 이미지를 처리하면 테이블에 행이 누적됩니다
6. CSV/Excel/JSON으로 내보내기할 수 있습니다

## 기술 스택

- **Tesseract.js v5** - 브라우저 OCR 엔진 (한국어 + 영어)
- **SheetJS** - Excel 파일 생성
- **Vanilla JS** - 프레임워크 없이 순수 JavaScript
- **GitHub Pages** - 정적 호스팅
